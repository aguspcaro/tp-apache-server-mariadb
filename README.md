# TP Integrador - Apache Server + MariaDB

Trabajo Práctico Integrador de Computación Aplicada - Universidad de Palermo.

## Descripción

Configuración de un servidor GNU/Linux Debian con los siguientes servicios:
- SSH con autenticación por clave pública/privada
- Servidor web Apache con soporte PHP
- Base de datos MariaDB
- Configuración de red estática
- Particionado de disco con montaje automático
- Script de backup automatizado con cron

## Contenido del repositorio

| Archivo | Descripción |
|---|---|
| `root.tar.gz` | Directorio /root |
| `etc.tar.gz` | Directorio /etc |
| `opt.tar.gz` | Directorio /opt (incluye scripts) |
| `www_dir.tar.gz` | Directorio /www_dir |
| `backup_dir.tar.gz` | Directorio /backup_dir |
| `var_part_aa`, `var_part_ab` | Directorio /var spliteado en partes |
